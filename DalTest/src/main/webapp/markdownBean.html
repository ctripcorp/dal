AutoMarkUpBatches

MarkUp尝试放量的次数，如果为不大于0的整数，达到AutoMarkUpDelay的值时，则直接MarkUp; AutoMarkUpBatches大于0时，是否Mark up以放量的批次(AutoMarkUpSchedule)为准
Set AutoMarkUpVolume to 0

AutoMarkUpDelay

秒，AutoMarkUpBatches不大于0时，自动MarkDown后，120秒开始MarkUp。
Set AutoMarkUpDelay to 30 seconds
